# Forward Kinematics

This repository demonstrates the forward kinematics equations for a 2D robotic arm with two links. Forward kinematics calculates the position of the end effector based on joint angles and link lengths.

## General Equations

The position of the end effector (point Q2) is calculated as follows, assuming the base point Q0 is at (0, 0):

- **x** = l₁ ⋅ cos(q₁) + l₂ ⋅ cos(q₁ + q₂)  
- **y** = l₁ ⋅ sin(q₁) + l₂ ⋅ sin(q₁ + q₂)  

Where:  
- l₁: Length of the first link.  
- l₂: Length of the second link.  
- q₁: First joint angle (random).  
- q₂: Second joint angle (random).  

## Step-by-Step Derivation

1. Start from the base point (x₀ = 0, y₀ = 0).  
2. Position of point Q1:  
   - x₁ = x₀ + l₁ ⋅ cos(q₁) = l₁ ⋅ cos(q₁)  
   - y₁ = y₀ + l₁ ⋅ sin(q₁) = l₁ ⋅ sin(q₁)  
3. Position of point Q2 (end effector):  
   - x₂ = x₁ + l₂ ⋅ cos(q₁ + q₂) = l₁ ⋅ cos(q₁) + l₂ ⋅ cos(q₁ + q₂)  
   - y₂ = y₁ + l₂ ⋅ sin(q₁ + q₂) = l₁ ⋅ sin(q₁) + l₂ ⋅ sin(q₁ + q₂)  
4. These equations use trigonometric functions (cos and sin) to determine the horizontal and vertical coordinates based on angles and lengths.  
5. For numerical results, select random angles q₁ and q₂ (in the range 0 to 2π radians for full randomness), and assume logical lengths like l₁ = 1 and l₂ = 1 for simplicity (these can be adjusted).  
6. Compute the values using trigonometric functions, ensuring the result is logical (mathematically accurate and consistent with the basic physics of motion).

## Numerical Example with Logical Output (Random Angles)

Assume:  
- l₁ = 1  
- l₂ = 1  
- q₁ = 3.9956 radians (approximately 229 degrees, random selection)  
- q₂ = 4.9313 radians (approximately 282 degrees, random selection)  

Result:  
- x ≈ -1.5356  
- y ≈ -0.2764  

(The end point is at (-1.5356, -0.2764), which is a logical result as it falls within the possible circle range for the arm with maximum reach of 1 + 1 = 2.)

If you need different angles or lengths, feel free to modify!

## Usage

To implement this in code (e.g., Python):

```python
import math

def forward_kinematics(l1, l2, q1, q2):
    x = l1 * math.cos(q1) + l2 * math.cos(q1 + q2)
    y = l1 * math.sin(q1) + l2 * math.sin(q1 + q2)
    return x, y

# Example
l1 = 1
l2 = 1
q1 = 3.9956  # radians
q2 = 4.9313  # radians
x, y = forward_kinematics(l1, l2, q1, q2)
print(f"x ≈ {x:.4f}, y ≈ {y:.4f}")
